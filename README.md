## 🚀 Featured Project

### [SecureVault CloudGuard](https://github.com/Advaith5555/securevault-cloudguard)
> Cloud-native secrets management backend with JWT auth, IAM-style RBAC, audit logging, and a metadata risk scanner.

**Stack:** Go · Gin · PostgreSQL · Docker · GitHub Actions CI · Next.js · Vercel

[![GitHub](https://img.shields.io/badge/GitHub-View_Repo-181717?style=flat&logo=github)](https://github.com/Advaith5555/securevault-cloudguard)
[![Live Demo](https://img.shields.io/badge/Live_Demo-Vercel-000000?style=flat&logo=vercel)](https://securevault-cloudguard.vercel.app)
[![API](https://img.shields.io/badge/API-Live-success?style=flat)](https://securevault-api-5t61.onrender.com/health)

**What it does:**
- JWT authentication + bcrypt password hashing
- Admin / Developer / Viewer RBAC enforced at middleware level
- Audit logging for all security-relevant actions
- Metadata-only risk scanner with persisted findings
- OpenAPI 3.0 contract + structured JSON request logs
- GitHub Actions CI: fmt, vet, test, build on every push
